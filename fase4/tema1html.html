Fundamentos de HTML

HTML para todos

¿Para qué se usa HTML?


Imagen de pikisuperstar en Freepik
HTML se utiliza para crear la estructura de las páginas web que se muestran en la World Wide Web (www). Contiene etiquetas y atributos que se utilizan para diseñar las páginas web. Además, podemos vincular varias páginas mediante hipervínculos (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

Estructura de HTML

Etiquetas

Las etiquetas HTML se utilizan para estructurar el contenido del sitio web (texto, hipervínculos, imágenes, medios, etc.). Las etiquetas no se muestran en los navegadores, solo "instruyen" a los navegadores sobre cómo mostrar el contenido de la página web. Todas las etiquetas están encerradas entre llaves angulares . Excepto por algunas etiquetas, la mayoría de las etiquetas comienzan con llaves angulares y se cierran con las llaves angulares correspondientes (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

!DocType Html

Es una instrucción que le dice al navegador web sobre el lenguaje de marcado en el que está escrita la página actual. No es un elemento o etiqueta. La declaración de doctype no distingue entre mayúsculas y minúsculas (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

html

Esta etiqueta le dice al navegador que es un documento HTML. Es el segundo elemento contenedor externo que contiene todos los demás elementos dentro de él (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

Encabezado (head)

Esta etiqueta se utiliza para definir la parte del encabezado del documento HTML que contiene información relacionada con el documento. Los elementos dentro de la etiqueta principal no son visibles en el front-end de una página web (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

Titulo de la página (title)

Cada documento tiene al menos un título. En la captura de pantalla anterior, se puede ver que la sección del título comienza con y termina con y en el medio se coloca el nombre del título de un documento según su elección. Desde arriba, se entiende que uno necesita abrir las llaves angulares, cerrar las llaves angulares y poner el nombre del título en el medio. Esta apertura y cierre angulares se aplican a casi todas las etiquetas en html (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).

Cuerpo del documento (body)

La etiqueta del cuerpo se utiliza para encerrar todo el contenido visible de una página web. En otras palabras, el contenido del cuerpo es lo que el navegador mostrará en el front-end. Un documento HTML se puede crear utilizando cualquier editor de texto. Guarde el archivo de texto usando .html o .htm . Una vez guardado como documento HTML, el archivo se puede abrir como una página web en el navegador (GeeksforGeeks, 2022; Pedamkar, s.f.; w3docs, s.f.).
